# Drum Samples

Place your drum sample files here:

- `kick.wav` - Kick drum sample
- `snare.wav` - Snare drum sample
- `hihat.wav` - Hi-hat sample
- `clap.wav` - Clap sample
- `tom.wav` - Tom drum sample

## Recommended Sources

1. **Freesound.org** - Search for "drum kick", "drum snare", etc. Filter by CC0 or CC-BY license
2. **Zapsplat** - Free sound effects library (requires free account)
3. **Sample Focus** - Free sample library

## File Format

- Format: WAV or MP3 (WAV preferred for quality)
- Sample rate: 44.1kHz recommended
- Bit depth: 16-bit or 24-bit
- Duration: 0.5-2 seconds per sample

## Quick Start

If you need samples quickly, you can use any short drum sound files and rename them to match the expected filenames above.
